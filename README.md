# Retail-Strategy-Analytics-Project-Quantium-Simulation

## Project Overview

This project simulates a real-world retail analytics scenario provided by Quantium. The objective was to analyse customer transaction data and generate actionable insights to support strategic decision-making in a supermarket chips category.

The project is divided into two key parts:

* **Task 1:** Exploratory Data Analysis & Customer Segmentation
* **Task 2:** Store Trial Evaluation & Impact Analysis

## Business Problem

The goal was to understand customer purchasing behaviour and evaluate the effectiveness of a store trial aimed at increasing sales performance.

Key questions addressed:

* Who are the most valuable customer segments?
* What factors drive chip sales?
* Did the store trial lead to a measurable uplift in performance?

## Key Insights

* Identified key customer segments contributing the highest sales (e.g., based on lifestage and spending behaviour)
* Found that premium customers and certain demographics drive higher basket value
* Revealed purchasing patterns such as frequency vs spend trade-offs
* Conducted trial vs control store comparison to assess sales uplift
* Evaluated whether observed changes were statistically meaningful

## Skills & Tools

* **Data Analysis:** Exploratory Data Analysis (EDA), data cleaning
* **Analytics Techniques:** Customer segmentation, benchmarking, trial evaluation
* **Tools:** R (tidyverse), Excel
* **Data Visualization:** ggplot2 / charts for business insights
